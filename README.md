# Lucientness

![Build](https://img.shields.io/badge/build-passing-brightgreen?style=for-the-badge) ![License](https://img.shields.io/badge/license-MIT-blue?style=for-the-badge) ![Version](https://img.shields.io/badge/version-1.2.0-orange?style=for-the-badge)

Lucientness adalah aplikasi editor kode untuk Android yang dikembangkan dengan antarmuka bergaya dark fantasy. Aplikasi ini dirancang dengan pendekatan desain minimalis, kaku (boxy), dan menghindari elemen visual yang berlebihan. Tujuan utamanya adalah memberikan pengalaman menulis kode yang bersih, fokus, dan didukung oleh asisten kecerdasan buatan.

## Fitur Utama

- **Dukungan Berbagai Bahasa Pemrograman**: Editor ini mendukung berbagai bahasa pemrograman dan markup, termasuk HTML, CSS, JavaScript, TypeScript, XML, Markdown, Python, Java, C, Ruby, Perl, Scala, dan PHP.
- **Pratinjau Langsung**: Tersedia fitur pratinjau instan untuk file HTML, CSS, JavaScript, dan Markdown langsung di dalam aplikasi.
- **Asisten AI Terintegrasi**: Dilengkapi dengan asisten cerdas yang memahami konteks file yang sedang dikerjakan. Asisten ini mendukung konfigurasi kustom untuk endpoint OpenAI-compatible, memungkinkan pengguna memakai model bahasa sesuai preferensi.
- **Manajemen Berkas Lokal**: Pengguna dapat membuat, menghapus, serta mengelola file dan direktori secara langsung melalui navigasi internal.

## Panduan Instalasi

Untuk membangun aplikasi ini dari kode sumber, pastikan Anda telah memasang Android Studio dan Android SDK terbaru.

1. Kloning repositori ini:
   git clone https://github.com/shawawah12-alt/lucientness.git

2. Buka proyek menggunakan Android Studio.

3. Tunggu hingga proses sinkronisasi Gradle selesai.

4. Jalankan aplikasi pada emulator atau perangkat fisik Android.

## Konfigurasi Asisten AI

Lucientness memberikan kebebasan dalam penggunaan asisten pintar. Secara bawaan, aplikasi ini memerlukan konfigurasi manual sebelum asisten dapat digunakan.

1. Buka aplikasi dan akses menu asisten cerdas melalui ikon bintang di bagian atas.
2. Masukkan Endpoint URL yang kompatibel dengan format OpenAI.
3. Masukkan API Key yang valid.
4. Tentukan nama model yang ingin digunakan.
5. Simpan pengaturan, dan asisten siap membantu Anda menulis kode.
